# Brand Assets

Place your company's logo files here.

## Required

- **White SVG logo** — Used by the image generator for banner overlays and by the PDF builder for document headers.
  - Filename must match the `LOGO_FILENAME` value in your `.env` file
  - Default expected: `company-logo-white.svg`

## Example

```
context/assets/company-logo-white.svg
```

With `.env`:
```
LOGO_FILENAME=company-logo-white.svg
```
